# 데이터셋 디렉토리

이 디렉토리는 프로젝트에서 사용하는 데이터셋을 저장하는 곳입니다.

## 샘플 데이터 생성

테스트용 샘플 데이터를 생성하려면:

```bash
python src/download_datasets.py
```

선택 옵션 1을 선택하면 다음 파일들이 생성됩니다:
- `sample_users.csv`
- `sample_events.csv`
- `sample_campaigns.csv`

## 데이터셋 다운로드

실제 데이터셋은 프로젝트 루트의 `README.md` 파일의 "데이터셋" 섹션을 참고하세요.

## 주의사항

- 이 디렉토리의 `.gitkeep` 파일은 Git에 빈 디렉토리를 유지하기 위한 것입니다.
- 실제 데이터 파일은 `.gitignore`에 의해 Git에서 제외됩니다.
- 데이터 파일을 공유할 때는 개인정보 보호 규정을 준수하세요.
